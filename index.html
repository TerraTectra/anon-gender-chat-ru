Поиск без лишних условий

Бот подходит для быстрых знакомств и обычного общения. Можно искать случайного собеседника или включить фильтры по полу и возрасту.

  • Рандомный поиск доступен без лимита.
  • Поиск по полу и возрасту сначала бесплатный.
  • Лимит тратится только когда собеседник найден.
  • Информация о собеседнике скрыта без лишнего раскрытия данных.

Фильтры

Выбирайте пол и возраст собеседника. В сутки доступно 50 успешных фильтрованных поисков.

Анонимность

Сообщения пересылаются без раскрытия аккаунтов. Собеседники не видят личные данные друг друга.

Без рекламы

Нет обязательных рекламных заданий перед поиском. Открыл бота, настроил профиль, начал общение.

Безопасность 12+

Пользователи 12-17 общаются только с 12-17, пользователи 18+ только с 18+. Для нарушений есть жалоба и блокировка собеседника.

FAQ

Это чат рулетка в Telegram?

Да, можно искать случайного собеседника, но также есть фильтры по полу и возрасту.

Можно искать по полу бесплатно?

Да. Фильтрованный поиск доступен бесплатно в пределах ежедневного лимита.

Есть обязательная реклама?

Нет. Главная идея бота: поиск без обязательной рекламы и навязанных заданий.
