# Анонимный чат 12+ в Telegram

SEO-лендинг для Telegram-бота [@anon_gender_chat_ru_bot](https://t.me/anon_gender_chat_ru_bot?start=src_github_pages_readme).

Бот помогает найти собеседника в Telegram:

- анонимный чат один на один;
- рандомный поиск без лимита;
- поиск по полу и возрасту;
- без обязательной рекламы перед поиском;
- разделение 12-17 и 18+ для безопасного общения;
- жалобы, блокировка и закрытая админ-модерация.
