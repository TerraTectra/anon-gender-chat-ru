# Анонимный чат 12+ — бот

Готовый Telegram-бот для анонимного общения один на один. Работает в режиме long polling и хранит данные в локальной SQLite-базе, поэтому для первого запуска не нужен отдельный сервер базы данных.

## Возможности

- случайный поиск без лимита;
- фильтр по полу и возрасту, 50 успешных совпадений в сутки;
- строгое разделение пользователей 12–17 и 18+;
- анонимная пересылка текста, фото, видео, документов, голосовых, стикеров и кружков;
- следующий собеседник и завершение чата;
- взаимные блокировки и жалобы;
- закрытые команды администратора `/stats` и `/reports`;
- учёт источников по параметру `/start`;
- SQLite в WAL-режиме, без хранения переписки.

## Запуск на Windows

1. Установите Node.js 22 или новее.
2. Запустите `start.ps1`.
3. При первом запуске откроется `bot/.env`. Вставьте токен после `BOT_TOKEN=` и сохраните файл.
4. Запустите `start.ps1` повторно.

Для доступа к `/stats` и `/reports` укажите Telegram ID администраторов через запятую:

```dotenv
BOT_TOKEN=123456:example
ADMIN_IDS=123456789,987654321
DB_PATH=./data/chat.db
```

Токен и база исключены из Git. Бот не должен запускаться одновременно на двух компьютерах с одним токеном в режиме long polling.
